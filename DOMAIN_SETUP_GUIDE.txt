========================================
SOLUTION OPTIONS
========================================

OPTION 1: Configure Domain to Use Port 8080
-------------------------------------------
Update your DNS/domain settings to point to:
- Domain: automation-reports.mobilehmi.org:8080
- This will make the URL: http://automation-reports.mobilehmi.org:8080

OPTION 2: Port Forwarding (Recommended)
---------------------------------------
Set up a reverse proxy to forward port 80 → 8080:

For IIS (Windows):
1. Install IIS with Application Request Routing (ARR)
2. Create a reverse proxy rule:
   - Incoming: port 80
   - Forward to: localhost:8080
3. Domain will work on standard port 80

For Apache:
Add to httpd.conf:
```
ProxyPass / http://localhost:8080/
ProxyPassReverse / http://localhost:8080/
```

For Nginx:
```
server {
    listen 80;
    server_name automation-reports.mobilehmi.org;
    
    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

OPTION 3: Run Directly on Port 80
---------------------------------
Run as Administrator and use port 80:
1. Open Command Prompt as Administrator
2. cd "E:\report generator\SERVER"
3. set PORT=80
4. python deploy.py
